#WAgent

WAgent is a platform where user can create a whatsapp agent for themselves or use one of our prebuilt agents.

## Tech Stack

- React.js
- Express.js
- Tailwind CSS
- TypeScript
- Supabase
- Python
- FastAPI
- AWS
- Portia AI
- Baileys

## Setup

- Clone the repository
- Install dependencies
- Set up environment variables
- Run the server
- Run the client
